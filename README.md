# ESP32-P4 JC1060P470 Sci-fi Dashboard

This ESPHome project creates a futuristic sci-fi dashboard interface using an ESP32-P4 microcontroller with a JC1060P470 display module.

## Features
- Custom sci-fi themed user interface
- Real-time sensor data visualization
- ESP32-P4 powered for enhanced performance
- Compatible with Home Assistant integration

## Hardware
- ESP32-P4 microcontroller
- JC1060P470 display module
- Additional sensors (to be configured)

## Audio Support
Audio functionality has not been tested yet.

## Setup
1. Configure ESPHome with your WiFi credentials
2. Flash the firmware to ESP32-P4
3. Connect to Home Assistant (optional)

## Configuration
Update the ESPHome YAML configuration file with your specific sensor setup and display preferences.
